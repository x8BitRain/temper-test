# temper-test

Demo: https://x8bitrain.github.io/temper-test

A tech test involving a sortable list of posts with with a log of changes and the ability to revert to a previous state for every rearranged post.

Built using:
 - Vue 3
 - Vite
 - Vitest
 - TailwindCSS
 - Pinia
 - SCSS

### Building

``yarn build``

### Testing 

``yarn test``
